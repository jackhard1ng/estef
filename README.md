# Para Mi Princesa - Hello Kitty Valentine

Una pagina web especial de San Valentin con tema de Hello Kitty.

## Como agregar las fotos

Pon tus 3 fotos en la carpeta `images/` con estos nombres:
- `foto1.jpg`
- `foto2.jpg`
- `foto3.jpg`

## Como ver la pagina

Simplemente abre el archivo `index.html` en tu navegador.
